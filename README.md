# Personal Dictionary system with Flask-SQLAlchemy

### Installation & Run:

* This project uses [WordsAPI](https://www.wordsapi.com/), so `X-RAPIDAPI-KEY` must be set in .env file.

After the .env editing, simply run;
```
flask run
```
For the migrations;
```
flask db upgrade
```
